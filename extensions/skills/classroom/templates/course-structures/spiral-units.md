# Course Structure — Spiral Units

> **Summary (for offering to the user):** Break a long course (e.g. 52 weeks)
> into short thematic units (4–6 weeks) focused on one concept cluster each,
> and deliberately **revisit earlier concepts in new contexts** across later
> units. Best for foundational skills that need repetition to stick — early
> literacy, phonics, sight words, number sense.

## Shape

- Divide the year into units (4–6 weeks works well), each on a single concept cluster (e.g. "Letters A–F", "Blending CVC words", "Sight Words Set 1").
- Each unit feels like a fresh start (own cover, own focus) — no overwhelm.
- **Spiral:** earlier letters/words/skills reappear inside later units in new contexts, rather than being taught once and dropped. This is the core of the structure, not an optional extra.

## Worked example
52-week dyslexia-friendly English: ~9 units, each a concept cluster, with alphabet and high-frequency words spiralling throughout; LEGO blending built in.

## When to use
Foundational/remedial skills; dyslexic learners who need multi-sensory spaced repetition; younger children who do better with frequent fresh starts.
