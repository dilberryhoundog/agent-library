Standing constants for this family. They apply to every lesson build. A learner's own file
(`students/<name>.md`) may override a specific item.

## Fixed constants

- Spelling convention: <e.g. Australian English (colour, practise, organised, metre)>.
  Use smart quotes and em dashes.
- Page size on every printable: <e.g. A4>.
- Cost rule: <e.g. no paid subscriptions or apps; build custom materials and use free
  resources; any existing curriculum used must be free to obtain>.
- Student workbook and answer key are always separate documents. (Recommended default —
  keep unless you have a reason not to.)
- Printables double as tablet annotation templates: wide outer margin, dotted write-lines,
  large labelled drawing boxes, checkbox/circle answers, clean page breaks. <Note your
  annotation workflow, e.g. "Save HTML to PDF on the iPad (Share → Print → Save as PDF) and
  annotate in GoodNotes.">

## Standard deliverable set

A complete unit study usually comprises a scope & sequence, a parent lesson guide, the
student workbook, a separate answer key, and — at completion — optionally a certificate and
competency report. <Adjust to your standard set.>

## Shared learning preferences

<Starting points shared across all the children; individual specifics live in each student
file. Example for dyslexic, kinetic learners:>

- A visual worked example before any reading, at the top of each lesson.
- A short "watch first" video before written work where one exists.
- Instructions as single numbered steps; no walls of text.
- Varied question formats to hold attention.
- Hands-on / manipulative activities.
- Note-taking via skeleton outlines and charts, not blank lines.
- Confidence and reflection built into every unit.
