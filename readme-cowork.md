# Cowork Library

A custom marketplace of plugins for **Co-work** — skills converted from the
authoring repos into installable, versioned plugins so they can be used inside
Co-work.

## Marketplace Compatibility

This marketplace targets **Co-work**, and many of its plugins also work in
**Claude Code**. Some plugins are more suited to one of the two platforms. If
you come across any compatibility issues, please report them with an issue.

## Quickstart

Installing is the same idea on both platforms — register this marketplace, then
install whatever you want from it. Only the navigation differs.

**In Co-work** (Mac or Windows app)

1. Open the **Cowork** tab, then **Customise**.
2. Add a **Personal plugin** → **Create plugin** → **Add marketplace**.
3. Use the green **Code** button above to copy the URL.
4. Paste this repository's URL into the marketplace dialog.

**In Claude Code**

1. Run `/plugins`.
2. Open the **Marketplaces** tab and choose **Add marketplace**.
3. Use the green **Code** button above to copy the URL.
4. Paste this repository's URL into the marketplace dialog.

**Then, on either platform:** open the **Plugins** tab and install any — or all —
of the plugins on offer. Plugins update themselves as new versions ship.

## Delivery

Plugins are delivered and updated through Claude's official marketplace
mechanism — a single install serves both Claude Code and Co-work, and keeps
installed plugins current automatically as their versions change.

## Advantages

- **Cross-platform reach**, working across Claude platforms.
